@ is the universal tag across all discussions/chats to tag someone.

**Why:** Consistent with every modern collaboration tool (Slack, Notion, Figma, etc.). Freelancers and clients already know this pattern. No learning curve.

**How to apply:** Whenever building chat, comments, conversations, or any text input where users communicate — support @mentions with autocomplete showing workspace members, clients, and collaborators. The @ trigger should work in: conversation panel, comment panel, block editor inline, deliverable comments, and any future messaging features.
